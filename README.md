## 📝 About the Project

This project is an interactive computer simulator inspired by the iconic **CANDA (C&A NDA)** terminal seen in *The Amazing Digital Circus*. It is built natively using HTML and CSS to recreate the nostalgic, retro aesthetic of the show's digital environment.

> **⚠️ Project Status:** > This simulator is currently a **Work in Progress**. It has not yet fully replicated the original computer from TADC to a 1-to-1 scale, and active development is still ongoing to refine the interface and features.

## 🛠️ Built With

* **HTML5** for structure
* **CSS3** for styling and animations
